Power Rankings

How this works

Current power uses the same team blend as offensive rankings: 30% roster talent (2026 PA–weighted player ratings), 50% record (win % z-score), 20% strength of schedule (avg. opponent win %).

Season projections start from each team’s current W-L, then add expected wins and losses on every remaining game. That uses the same underlying matchup model as the matchup predictor, but with a sharper win curve (less pulled toward 50/50) so strong teams can project toward records like 18–4 and weak teams toward heavier losses. Projected win % keeps fractional expected wins; the W-L column rounds to whole numbers. Regular season length: 22 games.

157 of 157 unplayed schedule games were simulated (games missing roster data are skipped).
