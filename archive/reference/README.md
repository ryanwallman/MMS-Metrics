# Reference files (not loaded at runtime)

| File | Purpose |
|------|-------|
| `mms-bylaws.pdf` | League bylaws document. Matchup bench rules are encoded in `lib/matchupMissingPlayers.js`. |
| `stats2025.csv` | Optional local backup of 2025 stats. The app fetches live 2025 data from Google Sheets; path constant `CSV_2025_STATS` in `lib/dataPaths.js` points here if you need a disk copy. |
